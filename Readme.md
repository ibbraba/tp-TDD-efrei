# TP  Poker 

## Jeu
* Cette version comporte deux joueurs (l'utilisateur et un ordinateur)

* La duplication de cartes n'est pas possible (un seul deck est utilisé pour tirer les cartes)

* Le résultat de la partie est visible dans la console


## Tests

* Certains tests sont commentés car il nécessitent une implémentation du mock de la console mais montrent le procédé de test



## Environnement .NET

* Language utilisé C#
* Les fichiers sont répartis dans chaque dossier lié au use case  
* Pour lancer les tests à la racine du projet 
```
dotnet test
```

* Pour lancer le jeu sur la console 
```
dotnet run
```

Enjoy !
